Integrate DHL services into your business applications with a few easy steps.


Implementing DHL Integration Solutions

The Toolkit / Developer Guides has sample messages, utilities and documentation to assist you to create you own DHL services enabled application.
Once you receive the Toolkit / Developer Guides, you can access the Testing and Certification environment. Do ensure you have the development resources experienced with XML messages and Web Services to work with this Toolkit / Developer Guides.
Request certification by providing the test case results. Once these are validated by a DHL representative, you are certified as production-ready.
You will receive access to production environment and can start using your application with DHL Integration Solutions.
